  # 🏦 Bank Management System (Java)

## 📌 Overview

The **Bank Management System** is a console-based Java application that simulates basic banking operations.
It allows users to perform transactions like deposit, withdrawal, and balance checking through a simple interface.

---

## 🚀 Features

* 🆕 Account Creation
* 💰 Deposit Money
* 💸 Cash Withdrawal
* ⚡ Fast Cash
* 📊 Balance Inquiry
* 🧾 Mini Statement
* 🔐 PIN Change
* ❌ Exit

---

## 🛠️ Technologies Used

* Java (Core Java)
* Object-Oriented Programming (OOP)

---

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/bank-management-system.git
```

### 2. Open project folder

```bash
cd bank-management-system
```

### 3. Compile the program

```bash
javac Main.java
```

### 4. Run the program

```bash
java Main
```
